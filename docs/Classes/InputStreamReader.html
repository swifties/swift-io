  • read()

    Reads next String from the stream. Max String size retrieved is influenced by buffer size and encoding set in the initializer.

    Returns

    Next string from the stream. Empty String can be returned from the reader - which means more data will be read from the stream on next iteration of read(). Nil when at the end of the stream.
