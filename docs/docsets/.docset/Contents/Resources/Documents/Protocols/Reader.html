Reader

public protocol Reader: Closeable

Reader interface Reader will return next string from the source. Size of the string returned depends on the Reader implementation.

  • read()

    Attempts to read string from the Reader

    Returns

    Next string from the source. Size of the string is dependent on the implementation of the reader. Return value can be empty string which would mean Reader is waiting for more data to come. Nil when at the end. Multiple reads from the end of the reader will all return nil.

    Throws

    Exception if read error occurs or if the Reader is already closed
